# `@savejs/npm`

Save your npm package namespace.

## Install

```bash
pnpm -g add @savejs/npm
```

```bash
yarn global add @savejs/npm
```

```bash
npm i -g @savejs/npm
```

## Usage

```bash
save-npm [@scope/]<package>[@version]
```

### Examples

```bash
save-npm @savejs/npm@0.0.1
 > ERR 403 Forbidden ...
 
save-npm save-js-npm@0.0.1
 > ... npm publish save-js-npm@0.0.1
```
