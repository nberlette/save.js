# save.js

Save your npm package namespace.

## Install

```bash
pnpm -g add save.js
```

```bash
yarn global add save.js
```

```bash
npm i -g save.js
```

## Usage

```bash
save.js [@scope/]<package>[@version]
```

### Examples

```bash
save.js awesome-pkg@0.0.1

 > ... npm publish awesome-pkg@0.0.1
```
